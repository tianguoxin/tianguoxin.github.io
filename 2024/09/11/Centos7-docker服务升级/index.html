Centos7 docker服务升级

DaXin Space
2024-09-11
编程

起因是在使用docker拉取镜像时遇到了一个错误：

通过yum search搜索发现是docker服务的版本太低了，yum提供的docker版本是1.13.1

升级方法如下：

  1. 删除历史docker服务
  2. 安装yum-utils
  3. 添加yum源
  4. 安装docker相关服务
  5. 启动docker服务

如果在使用docker-compose up -d命令时出现client version 1.22 is too old. Minimum supported API version is 1.24, please upgrade your client to a newer version错误，需要将docker-compose.yaml中的version:2.0修改为2.1
